## E-Commerce-Sales

## Scope of the project 
This project presents end-to-end sales analysis of a fictional e-commerce business using Microsoft Excel.

## Structure
After having load raw data ,we transformed each of these data into tables and loaded them in power pivot .
Then we connected the tables through Primary and Foreing Keys in order to create a total schema for Pivot use.

After that we were able to answer - in separate sheets - some queries.

In the end we created a dashboard presented all important info .
This dashboard represents : 
 1. KPIs
 2. Slicers
 3. Charts

## Business Questions 

1. How many products are available in each category?
2. Who are the highest-value customers?
3. Which customers place the highest number of orders?
4. Which product categories generate the highest revenue?
5. How has revenue changed over time?
6. Which products generate the highest revenue?

 ## Key Insights

- Total Revenue reached **$1,900**
- Electronics generated the highest revenue.
- Laptop was the best-performing product.
- John Smith generated the highest customer revenue.
- Revenue declined steadily from January to March.
- Active customers represented the majority of registered customers.
